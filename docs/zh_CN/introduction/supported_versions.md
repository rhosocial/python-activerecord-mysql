# 支持版本

## MySQL 版本支持

| MySQL 版本 | 支持状态 | 说明 |
|-----------|---------|------|
| 5.6.x | ✅ 支持 | 部分功能可能存在差异，如不支持 JSON 类型 |
| 5.7.x | ✅ 支持 | 推荐用于遗留系统 |
| 8.0.x | ✅ 推荐 | 当前主流稳定版 |
| 8.4.x | ✅ 支持 | 最新 LTS 版本 |
| 9.0.x | ✅ 支持 | 最新稳定版 |
| 9.6.x | ✅ 支持 | 最新次要版本 |

⚠️ **注意**：

- MySQL 5.6 不支持 JSON 数据类型，相关功能无法使用
- 部分功能在不同版本间可能存在细微差异，具体请参阅各功能文档

## MariaDB 支持

| MariaDB 版本 | 支持状态 | 说明 |
|-------------|---------|------|
| 10.x | ⚠️ 部分支持 | 仅支持与 MySQL 特性兼容的部分，未进行充分测试 |

⚠️ **注意**：MariaDB 仅支持与 MySQL 特性兼容的部分，部分 MySQL 特有功能可能无法正常工作。生产环境建议使用 MySQL。

## 依赖要求

| 依赖包 | 版本要求 | 说明 |
|-------|---------|------|
| rhosocial-activerecord | >=1.0.0 | 核心库 |
| mysql-connector-python | >=8.0.0 | MySQL 驱动（唯一支持）|

⚠️ **重要**：本后端仅支持 mysql-connector-python 驱动，不支持 mysqlclient、PyMySQL 等其他驱动。
